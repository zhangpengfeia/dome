# 03-js基础



##循环结构

###while循环

+ 当条件为true时，则开启无限月读模式，为flase时不执行。

###do-while循环

+ 先执行循环体后在进行判断，所以至少执行一次。

###for循环

+ 语法：

  ```javascript
  for(var i = 0;i<=10;i++){  }
  ```

+ 初始化表达式只执行一次，然后执行条件表达式，如果条件表达式返回结果为false则结束循环，最后执行自增

## break和continue

+ break : 中止循环
+ continue: 中止本次循环





##数组

+ 概念

  > 管理大量数据，有位置，顺序，下标，索引

+ 创建

  ```js
  1.字面量创建
  var arr = []
  2.简洁
  var arr2 = new Array()
  3.
  var arr3 = new Array(3) //规定长度
  ```

+ 下标

  ```js
  1.获取下标为0
  arr[0]
  2.赋值
  arr[0] = 值
  ```

+ 数组长度

  ```js
  arr.length
  ```
