# 简单与复杂类型

+ js数据都是存在内存中的，分别在两个地方存贮，堆和栈

+ 简单的类型会存入栈不会涉及堆

  ```js
  var a =	1;
  var b = a;
  b = 2;
  console.log(a,b) //a为1，b为2
  ```

  

+ 复杂类型的储存会存入栈设计堆

  ```js
  var obj1 = { a:1 };
  var obj2 = obj1;
  obj2.a = 2;
  
  console.log(obj1.a,obj2.a) // 2 2
  
  ```
