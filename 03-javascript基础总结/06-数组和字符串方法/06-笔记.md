##数组

###方法

```js
var arr = [1,2,3]
arr.push(4) //给数组后面添加一个元素
arr.pop() //从数组后面删除最后一个元素，返回值是被删除的元素 
arr.unshift(0) //从数组的最前面添加一个元素，返回的是原数组改变后的长度
arr.shift() //从数组前面删除一个元素，返回被删除的元素
arr.splice(3,1,4,2)//从下标为3的位置开始删除1个元素，在将4，2添加到数组中，返回值是数组，被删除的所有元素
arr.join('+'); //拆分数组，按照+号隔开，拼接为字符串
arr.indexOf(2) //查找数组中是否有此元素，有则返回下标，无返回-1
arr.findIndex(function(item){ return item>=20 }) //查找元素更加灵活


arr.some(function(i){ return i>5 }) //数组的某一个值大于5时返回true
arr.every(function(i){ return i>5 }) //数组每一个值大于5时返回true
```

###拼接

```js
var arr = [1,2,3]
var arr2 = [4,5,6];
concat(arr,arr2)  //将多个数组或数据拼接，返回拼的新数组，不改变原数组
arr.slice(1,4) //对数组进行截取,包左不包右，一般用于复制数组




```

##字符串方法

```js
var str = "1,2,3";
str.split(',') //将字符串用，号隔开拼接为数组
str.replace('b',a) //将匹配到的第一个字符串替换为a
str.indexOf() //查询字符串，有则返回字符串下标，无则返回-1
str.charAt()  //找到指定下标的那个字符，返回值是那个字符
str.substring(1,4) //截取字符串
str.trim() //去头尾空格
```
